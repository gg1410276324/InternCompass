# Product

## Register

product

## Users

Chinese university students exploring internship directions, plus reviewers who evaluate the project as a product-manager portfolio demo. Users need a calm desktop workspace where they can fill in a student profile, compare recommended internship directions, and inspect detailed role analysis without account setup or network dependencies.

## Product Purpose

Intern Compass is a local Python desktop tool that turns major, stage, interests, skills, constraints, and dynamic major questions into explainable internship direction recommendations. Success means the user can understand which role categories are worth prioritizing, why they fit, what skills are missing, and export a structured recommendation report.

## Brand Personality

Professional, clear, trustworthy, and lightweight. The product should feel like a polished SaaS-style analysis tool and product portfolio demo, not a campaign page, game interface, chat product, login system, or generic admin shell.

## Anti-references

Do not add login, registration, authorization, password gates, networking, a mobile layout, a web landing page, chat UI, or a decorative left icon navigation rail. Avoid cyberpunk, game-like visuals, heavy glassmorphism, loud gradients, poster-like hero sections, and dense enterprise clutter.

## Design Principles

- Preserve the three-column workbench: student profile, recommendation results, and job detail analysis.
- Make recommendations explainable at a glance through scores, levels, badges, and concise reasons.
- Keep the interface local-product credible: restrained color, readable hierarchy, stable controls, and no fake platform chrome.
- Prefer configuration-driven styling so future visual tuning happens in UI tokens rather than scattered widget code.
- Protect the existing data and rules boundary: visual polish must not alter recommendations, knowledge bases, search logic, exports, saved test data, or packaging.

## Accessibility & Inclusion

Use high-contrast body text, readable Chinese labels, consistent focusable controls, clear empty states, and scrollable long-detail areas. Motion should be minimal and nonessential because this is a desktop productivity tool.
